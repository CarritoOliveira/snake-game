# Jogo da Cobrinha 🐍

🌐 **Jogar Agora:** [Abrir Jogo](https://carritooliveira.github.io/nome-do-repositorio/)

Versão clássica e nostálgica do famoso **Jogo da Cobrinha**, desenvolvido em um único arquivo HTML com design retrô pixelado.

### 🎮 Como Jogar
- Use as **setas do teclado** para mover a cobra
- **Espaço** para pausar / continuar
- Coma as maçãs vermelhas para crescer e ganhar pontos
- Evite bater nas paredes ou no próprio corpo

### ✨ Funcionalidades
- Sistema completo de pontuação
- Recorde salvo no navegador (localStorage)
- Velocidade aumenta automaticamente conforme a pontuação
- Tela de Game Over com opção de jogar novamente
- Design retrô com fonte pixelada e efeitos neon
- Totalmente responsivo e otimizado para desktop

### 🛠️ Tecnologias Utilizadas
- **HTML5**
- **CSS3** (com animações e efeitos visuais)
- **JavaScript Vanilla** (lógica completa do jogo)

### 🎯 Objetivo do Projeto
Praticar:
- Lógica de programação com arrays e movimentação
- Manipulação do Canvas
- Controle de eventos do teclado
- Armazenamento local (`localStorage`)
- Organização de código em um projeto single-file
